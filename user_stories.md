# User Stories

```
Story 1

As a user I would like an app to view photos
So that I can view photos
I need the home page to show a list of photos
```
```
Story 2

As a user I want to contribute photos to the app
So that others can view my photos
I need to upload a photo
```
```
Story 3

As a user I want to be able to view my photos
So I can keep track of my uploads
I need to register and sign in
```
```
Story 4

As a user I want to be able to give feedback on other people's photos
So that I can show some appreciation of cool photos
I need to add a comment and a like
```
```
Story 5

As a user I want to customise my photos
So that I can add something new to the photo
I need to add a filter to the photo
```
